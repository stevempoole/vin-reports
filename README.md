# VIN Reports

Automated buyer's reports for used vehicles. Reports are generated by an agent
and published as static HTML on GitHub Pages.

Live site: https://stevempoole.github.io/vin-reports/

Each report combines NHTSA recall data, reliability research, market pricing,
and ownership cost analysis. Reports are public and not a substitute for a
Carfax/AutoCheck history check or a professional pre-purchase inspection.
